## General Tips

### For all cities
1. **Buy return ticket**: If using public transport, buy both ways
2. **Leave early**: Traffic unpredictable on match days
3. **Stay after**: Wait 30 min post-match for crowds to clear
4. **Meet point**: Agree location with friends in case separated
5. **Emergency cash**: Some vendors cash-only
6. **Phone charged**: For QR tickets and emergency calls
